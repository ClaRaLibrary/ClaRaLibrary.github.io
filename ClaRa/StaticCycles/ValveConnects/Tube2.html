Tube2

Created Dienstag 06 Juni 2017

A stationary model of a tube computing the outlet pressure.

1. Purpose of Model


The model is supposed to be used for the simplified simulation of static cases. Its main purpose is to provide appropriate start or nominal values for similar dynamic model versions. The tube model calculates the outlet pressure in dependence of the set load point. It considers frictional and geodatic pressure losses.

2. Level of Detail, Physical Effects Considered and Physical Insight


2.1 Level of Detail


Referring to Brunnemann et al. [1], this model refers to the level of detail L1 because the model only computes a set of parameters stationary in time for the given conditions.

2.2 Physical Effects Considered


  • Conservation of linear momentum (in steady flow)

3. Limits of Validity


The model is only able to calculate stationary situations.

6. Governing Equations


The governing equations consider mainly the pressure difference at the connection ports.

6.1 Governing Model Equations


The outlet pressure is computed from considering a frictional pressure loss and a geodatic pressure loss.

The fricional pressure loss is calculated with the nominal internal pressure loss and a user defined table which is dependent on the target power of the system in which the tube will be used.
